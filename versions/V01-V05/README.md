# V1-V5 — 探索期：趋势跟踪 → 均值回归

## 版本总览

| 版本 | 策略方向 | 回测收益 | 核心思路 |
|------|----------|----------|----------|
| V1 | 双均线交叉 | -3% | MA5/MA20 金叉做多，死叉做空 |
| V2 | 双均线 + ADX 趋势过滤 | -4% | 加 ADX>25 确认趋势强度 |
| V3 | ADX + ATR 止损 | -6% | ATR 动态止损，但趋势策略在 A 股震荡市频繁假突破 |
| V4 | 布林带均值回归 | +0.5% | 转向均值回归，价格触及下轨买入 |
| V5 | 布林带 + RSI 确认 | +2.2% | RSI 超卖确认布林带信号，减少假入场 |

## 核心教训

1. **趋势跟踪在 A 股水土不服**：A 股牛短熊长、震荡为主，双均线/ADX 趋势策略频繁止损
2. **均值回归更适合 A 股**：V4 转向均值回归后首次盈利，验证了超卖反弹的逻辑
3. **单一指标不可靠**：V5 加入 RSI 确认后收益提升，说明多指标共振优于单指标

## 策略逻辑概要

### V1-V3：双均线 + ADX 趋势跟踪

```
入场：MA5 上穿 MA20（金叉）
过滤：ADX > 25（确认趋势存在）
出场：MA5 下穿 MA20（死叉） / ATR 止损
```

### V4-V5：布林带均值回归

```
入场：价格触及布林带下轨(2σ)
过滤：RSI < 30（V5 新增，确认超卖）
出场：价格回到布林带中轨 / 止损 3%
```
