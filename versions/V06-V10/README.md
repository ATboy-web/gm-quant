# V6-V10 — 成长期：RSI 超卖反弹 → 精选股票池

## 版本总览

| 版本 | 策略方向 | 回测收益 | 核心思路 |
|------|----------|----------|----------|
| V6 | RSI(6) 超卖反弹 | +0.2% | RSI(6)<32 买入，RSI>70 卖出，首次尝试 RSI |
| V7 | RSI + 量价确认 | +5.3% | 加入放量确认，过滤无量反弹 |
| V8 | RSI + ATR 止损 | +25.7% | ATR 动态止损，保护利润 |
| V9 | 50 只股票池 + RSI 精选 | +11% | 扩大候选池，但过度分散 |
| V10 | RSI + 行业分散 | +32.4% | 行业分散持仓，降低系统性风险 |

## 核心教训

1. **RSI(6) 比 RSI(14) 更适合短线反弹**：6 日 RSI 更敏感，能更快捕捉超卖机会
2. **成交量确认极其重要**：V7 加入量价确认后收益大幅提升
3. **ATR 止损是利润保护的关键**：V8 用 ATR 替代固定百分比止损，收益从 5.3% 跳到 25.7%
4. **行业分散降低回撤**：V10 的行业分散策略显著降低回撤

## 策略逻辑概要

### V6-V8：RSI 超卖反弹

```
入场：RSI(6) < 32（超卖）
过滤：V7 新增量比 > 1.2（放量确认）
出场：RSI > 70（V6）/ ATR 跟踪止盈 2.0×（V8）
止损：固定 5%（V6-V7）/ ATR 止损 1.5×（V8）
```

### V9-V10：扩大股票池

```
股票池：V9 扩至 50 只（V8 约 20 只）
V10 新增：行业分散约束（单一行业最多 2 只持仓）
出场：RSI > 75 + ATR 跟踪止盈
```
