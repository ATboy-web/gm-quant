## V20 核心改进：反转确认策略(RT)

### 问题诊断
V19.2 的三大弱势行业（化工-3.7%、新能源-2.1%、煤炭-1.1%）共同特征：
**行业处于持续下跌趋势（左侧），MR策略的"超卖抄底"逻辑反复被套**

### 解决方案：RT 反转确认策略
不抄底，等反转确认后再入场（右侧交易）：
1. **MA5 > MA10**（短期趋势反转）
2. **放量确认**（反转日量比≥1.3）
3. **价格站上MA20**（确认脱离底部）
4. **RSI 40-60区间**（已有买盘介入，非超卖）
5. **近期有超卖历史**（确认是从底部反转，必要条件）
