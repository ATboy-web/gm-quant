# V17 — RSI(6) 均值回归 + MA60 出场

## 版本概要

| 项目 | 说明 |
|------|------|
| 策略方向 | RSI(6) 超卖均值回归 |
| 核心变化 | V16 基础上引入 MA60 趋势出场 + RSI 周期从 14 改回 6 |
| 回测收益 | **-19.02%** |
| 最大回撤 | -24.50% |
| 失败原因 | MA60 出场导致过早止损，大量持仓在反弹前被清掉 |

## V16 → V17 关键改动

1. **RSI 周期**：14 → 6（更敏感，但假信号增多）
2. **新增 MA60 出场**：收盘价跌破 MA60 → 强制清仓
3. **RSI 买入阈值**：30 → 32（更早入场）
4. **时间止损**：15 天 → 20 天
5. **入场评分门槛**：0.35 → 0.40（更严格）

## V17 出场优先级

MA60 趋势出场(跌破MA60) > ATR跟踪止盈(2.0x) > ATR止损(1.5x) > RSI过热(>82) > 时间止损(20天)

## 失败教训

MA60 出场是 V17 的致命缺陷：
- RSI 超卖买入的股票通常已经跌破 MA60，入场后很快触发 MA60 出场
- 形成"买入 → 立即跌破 MA60 → 止损 → 再买入"的死循环
- V18.1 的修复：关闭 MA60_EXIT_ENABLED，改用归属策略优先 + 止损安全阀的投票出场
