### 策略详解

| 策略 | 入场逻辑 | 出场逻辑 | 适用行业 | 回测贡献 |
|------|---------|---------|---------|---------|
| **MR** 均值回归 | RSI超卖+六因子打分≥阈值 | ATR跟踪止盈/止损/RSI过热/时间止损 | 全行业 | +21.8% |
| **MOM** 动量趋势 | MA20>MA60多头+放量突破 | 跌破MA20/死叉 | 全行业 | +1.7% |
| **VP** 量价背离 | 价新低+缩量=抛压耗尽 | 放量反弹/放量滞涨 | 全行业 | +2.9% |
| **BK** 突破策略 | N日价格突破+放量确认 | 跌破突破K线低点/ATR止损 | 高波动行业 | +19.8% |
| **DV** 红利策略 | 相对低位+稳定量+RSI合理 | RSI过热/时间止损/趋势破位 | 低波动行业 | +24.6% |

### 行业差异化配置（sector_config.py）

**高波动行业**（有色/科技/军工/新能源）→ MR + MOM + VP + BK
**中波动行业**（化工/医药/消费/汽车）→ MR + MOM + VP
**低波动行业**（金融/煤炭/公用事业/地产）→ MR + MOM + VP + DV

每个行业有独立的: 策略权重、RSI阈值、ATR倍数、止损参数、持仓天数上限。

## 出场决策层级

```
1. 归属策略喊 SELL → 直接卖
2. 任何止损类信号（含跌破MA20）→ 安全阀直接卖
3. 趋势/时间类出场 → 需交叉验证(≥2.0票)
   ※ MOM低置信度(confidence≤0.5)不计入交叉验证
```

## 冷却期

卖出后 3 天内不再买入同一股票，防止反复买卖磨损。

## 实盘部署指南

### 环境要求
- 掘金量化终端 (≥3.0)
- Python 3.x (GM内置)
- 掘金账户已登录

### 部署步骤
1. 打开掘金终端 → 研究页
2. 将以上 17 个 .py 文件全部复制到策略项目目录
3. 修改 `config.py` 中的 `ACCOUNT_ID` 为你的实盘账户
4. 修改 `main.py` 中的 `run()` 调用:
   ```python
   # 回测模式
   run(mode_backtest=True, ...)
   # 实盘模式
   run(mode_backtest=False)
   ```
5. 点击运行，观察日志输出

### 关键参数（实盘前确认）
- `INITIAL_CASH`: 初始资金（默认 25000）
- `MAX_POSITIONS`: 最大持仓数（默认 4）
- `COOLDOWN_DAYS`: 冷却期天数（默认 3）
- `STOP_LOSS_PCT`: 止损百分比（各行业独立配置）

### 风控要点
- 单票最大仓位: 25%（4持仓均分）
- 硬止损: 行业差异化（1.5%-5%）
- 日最大亏损: 建议设 3% 触发暂停交易
- 冷却期: 3天（防止反复买卖）
