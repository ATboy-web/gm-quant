## 情绪引擎 (7源爬虫)

```
新浪财经 → 财联社 → 东方财富 → 雪球 → 东财股吧 → 同花顺 → 华尔街见闻
                              ↓
                    关键词+极性 → 板块情绪 [-1,+1]
                              ↓
                    悲观冻结 | 中性正常 | 乐观加权
                              ↓
                          策略选股
```

数据缓存: `data/news/{板块}_{日期}.txt` (7天自动清理)

## 交易流程

```
每根日K:
  tracer.heartbeat()          ← 日志心跳
  get_sector_sentiment()      ← 情绪引擎
  check_exits()               ← 出场检查
  find_buy_candidates()       ← 情绪过滤 → 策略打分 → 投票 → 选股
  trace.buy() / trace.sell()  ← 日志记录
```
