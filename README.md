## 情绪引擎 (sentiment_engine.py)

```
新浪滚动新闻 → 财联社电报 → 东方财富板块
         ↓
   关键词+极性分析 → 板块情绪评分 [-1, +1]
         ↓
   悲观板块冻结  |  乐观板块加权  |  中性正常
         ↓
      策略选股 (executor)
```

数据缓存: `data/news/{板块}_{日期}.txt` (7天自动清理)

## 风控体系

- **ATR动态止损**: 低波动 1.5× / 高波动 2.0-2.5×
- **时间止损**: 15-25天 (行业差异化)
- **硬止损**: -4%~-8% (行业差异化)
- **冷却期**: 卖出后1天不买入同股
- **投票交叉验证**: 卖出需≥1.5票确认
- **情绪冻结**: 极端悲观板块跳过

## 股票池 (96只 / 12行业)

- **选股标准**: 主板 + 正净利润 + 日均成交>1亿 + 上市>2024
- **含ST**: 利润为正的ST标的
- **首推**: 公用事业(电力)、有色、消费
