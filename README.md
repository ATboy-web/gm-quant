### 三策略并行

| 策略 | 代号 | 核心逻辑 | 市场偏好 | 市场加权 |
|------|------|----------|----------|----------|
| 均值回归 | MR | RSI(14) 超卖六因子打分，ATR 跟踪止盈 | 震荡市 | ×3.0(震荡) / ×2.0(牛) / ×1.5(熊) |
| 动量趋势 | MOM | MA20>MA60 多头排列 + 放量突破追涨 | 牛市 | ×2.0(牛) / ×0.5(震荡) / ×0.3(熊) |
| 量价背离 | VP | 价新低 + 缩量 = 抛压耗尽，抄底信号 | 熊市 | ×2.0(熊) / ×1.0(其他) |

### 投票融合规则

**入场决策矩阵**（MR 始终 ×2 权重）：

| BUY 加权票数 | SELL 票数 | 决策 | 仓位 |
|-------------|----------|------|------|
| ≥4.0 | 0 | BUY | 100%（MR + 双确认） |
| ≥2.0 | 0 | BUY | 70%（MR 主导） |
| ≥1.5 | 0 | BUY | 50%（MR 弱确认） |
| 1.0~1.5 | 0 | HOLD | 0%（非 MR 不可靠） |

**出场决策层级**（V18.1 核心改进）：

1. **归属策略优先**：持有股票的归属策略发出 SELL → 直接执行
2. **止损安全阀**：含止损关键词（`止损`/`stop`/`STP`）→ 无条件执行
3. **强卖出**：sell_votes ≥ 2.5 → 多数策略确认卖出
4. **交叉验证卖出**：sell_votes ≥ 1.5 + 非归属策略也确认 → 交叉验证通过

## 策略详解

### MR 均值回归（主导策略）

**六因子评分体系**：

| 因子 | 权重 | 说明 |
|------|------|------|
| RSI 超卖深度 | 35% | RSI(14) < 30，越超卖分越高 |
| 趋势对齐 | 20% | 价格在 MA20 上方 |
| 成交量确认 | 15% | 放量 ≥ 20 日均量 1.2 倍 |
| 近期跌幅 | 15% | 5%~20% 跌幅为最佳反弹区间 |
| 反转形态 | 8% | 看涨吞没 / 锤子线识别 |
| 行业动量 | 7% | 优选强势行业中的超卖股 |

**入场条件**：
- 标准入场：六因子总分 ≥ 0.35（熊市门槛 0.45）
- 极端超卖快通道：RSI < 18 直接触发（不依赖六因子）

**出场优先级**：ATR 跟踪止盈(2.5×) > ATR 止损(1.5×) > RSI 过热(>80) > 时间止损(15 天)

### MOM 动量趋势

**入场条件**：
- MA20 > MA60 多头排列 + 价格站上 MA20 + 综合评分 ≥ 0.50
- 评分：价差(40%) + 涨幅(35%) + 量比(25%)
- 金叉加分：+0.15

**出场条件**：跌破 MA20 / 死叉 / 止损 6% / 止盈 15% / RSI 极端 >85

### VP 量价背离

**入场条件**：
- 20 日内价新低 + 缩量（量比 < 0.65）+ 综合评分 ≥ 0.50
- 评分：量缩程度(40%) + 价格偏离(35%) + 超卖信号(25%)

**出场条件**：放量反弹结束 / 放量滞涨 / 止损 5% / 止盈 8% / 时间止损 20 天

## KNN 历史匹配引擎

当三策略投票出现分歧时，通过 KNN 搜索历史相似时刻来仲裁：

- **8 维特征向量**：RSI、MA20 偏离、MA60 偏离、量比、波动率、5 日收益、20 日收益、20 日回撤
- **加权欧氏距离**：按因子重要性赋权（RSI 权重最高）
- **推荐逻辑**：K=5 邻居中，检查各策略在历史相似时刻的胜率，推荐最优策略

## 使用方法

### 1. GM 终端回测

在掘金量化终端中打开 `main.py`，配置 `config.yaml` 中的回测参数，点击"回测"。

### 2. 离线验证

```bash
python3 offline_backtest.py
```

不依赖掘金终端，直接用 `history()` API 拉取数据模拟回测。

### 3. 推送代码到 GitHub

```bash
python3 push_to_github.py
```

通过 GitHub REST API（Git Data API）推送文件，绕过本地 git SSL/DLL 问题。

## 策略迭代历史

| 版本 | 策略方向 | 回测收益 | 关键变化 |
|------|----------|----------|----------|
| V1-V3 | 双均线 + ADX 趋势跟踪 | -3% ~ -6% | 初始探索，趋势策略在 A 股水土不服 |
| V4-V5 | 布林带均值回归 | +0.5% ~ +2.2% | 转向均值回归，初见成效 |
| V6-V8 | RSI(6) 超卖反弹 | +0.2% ~ +25.7% | 找到超卖反弹方向 |
| V9-V10 | 50 只池 + RSI 精选 | +11% ~ +32.4% | 扩大候选池 |
| V11-V12 | 分档建仓 / 动量突破 | 回退 | 过度优化导致回撤 |
| V13 | 回归简单 RSI | **+50.0%** | 删繁就简 |
| V14 | 移动止盈 + 60 只池 | **+56.0%** | 历史最佳 |
| V15 | 行业分散 50 只 | 回退 | 分散过度，信号稀释 |
| V16 | 六因子 + ATR + 市场自适应 | 迭代中 | 引入多因子框架 |
| V17 | RSI(6) 均值回归 + MA60 出场 | -19.02% | MA60 出场导致过早止损 |
| **V18.1** | **三策略融合 + MR 主导投票** | **+9.95%** | 多策略投票 + 归属策略优先出场 |

### V17 → V18.1 核心改进

1. **三策略并行**：从单一 RSI 均值回归扩展为 MR + MOM + VP 三策略互补
2. **投票融合**：多数策略确认才执行，过滤假信号
3. **MR 主导权重**：MR 作为核心策略始终 ×2 权重，回测胜率 66.7%
4. **归属策略优先出场**：修复 V17 的 MA60 过早止损问题
5. **市场状态自适应**：根据牛/震荡/熊动态调整策略权重
6. **RSI 周期调整**：RSI(6) → RSI(14)，平滑信号减少假突破
7. **KNN 历史匹配**：分歧时查历史相似时刻仲裁
