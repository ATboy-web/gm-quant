### 策略详解

| 策略 | 入场逻辑 | 出场逻辑 | 适用行业 |
|------|---------|---------|---------|
| **MR** 均值回归 | RSI超卖+六因子打分+趋势过滤 | ATR跟踪/止损/RSI过热/时间 | 全行业 |
| **MOM** 动量趋势 | MA20>MA60多头+放量突破 | 跌破MA20/死叉 | 趋势行业 |
| **VP** 量价背离 | 价新低+缩量=抛压耗尽 | 放量反弹/放量滞涨 | 周期行业 |
| **BK** 突破策略 | N日新高+放量确认 | 突破失败/止损/止盈 | 高波动行业 |
| **DV** 红利策略 | 相对低位+量稳+RSI40-55 | 止损5%/止盈10%/时间30d | 低波动行业 |
| **RT** 反转确认 | MA5>MA10+站上MA20+放量+底部超卖 | 死叉/跌破MA20/止损6% | 左侧行业 |

## V21 vs V20 关键改进

1. **GM主程序同步**: main.py从V19.2升级到V21，加入RT策略
2. **公用事业优化**: MR 4.0→2.5, RSI 38→34, 新增RT, 仓位0.35→0.30
3. **化工优化**: RT成为主导(2.5), MR降权(1.5)
4. **制造优化**: 新增RT策略
5. **趋势过滤**: MR策略加入MA20<MA60+行业动量为负时拒绝买入
6. **CSV实盘数据诊断**: 识别出GM运行版本与离线版本不匹配问题
