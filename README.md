# GM V16 — 多因子自适应均值回归策略

基于掘金量化（GoldMiner）平台的 A 股量化交易策略，迭代至第 16 版。

## 策略概览

| 项目 | 说明 |
|------|------|
| 策略类型 | 多因子自适应均值回归 |
| 交易市场 | A 股主板（SHSE 600/601/603，SZSE 000/001/002） |
| 数据频率 | 日线 |
| 回测区间 | 2024-01-01 ~ 2026-05-22 |
| 初始资金 | 25,000 元 |
| SDK 版本 | GM 3.0.183 |

## 核心逻辑

### 入场（六因子综合评分 ≥ 0.40）

| 因子 | 权重 | 说明 |
|------|------|------|
| RSI 超卖深度 | 30% | RSI(6) < 32，越超卖分越高 |
| 趋势对齐 | 20% | 价格在 MA20 / MA60 上方 |
| 成交量确认 | 15% | 放量 ≥ 20日均量 1.2 倍 |
| 近期跌幅 | 15% | 5%~20% 跌幅为最佳反弹区间 |
| 反转形态 | 10% | 看涨吞没 / 锤子线识别 |
| 行业动量 | 10% | 优选强势行业中的超卖股 |

### 出场（四重风控）

| 条件 | 参数 |
|------|------|
| ATR 动态止损 | 1.5× ATR，上限 8% |
| ATR 跟踪止盈 | 2.0× ATR，从最高点回撤触发 |
| RSI 过热 | RSI(6) > 82 |
| 时间止损 | 持仓 20 天未盈利强制退出 |

### 市场自适应

| 市场状态 | RSI 阈值 | 最大持仓 | 止损 |
|----------|----------|----------|------|
| 牛市 | 34 | 4 只 | 1.8× ATR |
| 震荡 | 32 | 4 只 | 1.5× ATR |
| 熊市 | 28 | 2 只 | 1.2× ATR |

### 仓位管理

- 行业分散：每行业最多 1 只
- 波动率调整：高波动行业仓位 ×0.7，低波动 ×1.2
- 单只仓位：约 30% 可用资金

## 股票池

35 只精选主板股票，覆盖 12 个行业：

金融 / 消费 / 医药 / 科技 / 新能源 / 制造 / 有色 / 汽车 / 化工 / 公用事业 / 军工 / 煤炭

详见 `stock_pool.py`

## 项目结构

```
├── config.py            # 策略参数配置（权重/阈值/市场状态）
├── indicators.py        # 技术指标纯函数库（RSI/ATR/MA/量比/形态）
├── stock_pool.py        # 35只精选股票池管理（12行业）
├── screener.py          # 六因子选股打分引擎
├── main.py              # 策略主程序（GM 回测入口）
└── offline_backtest.py  # 离线回测脚本（不依赖 GM 终端）
```

## 使用方法

### 1. GM 终端回测

在掘金量化终端中打开 `main.py`，点击"回测"。

### 2. 离线验证

```bash
python offline_backtest.py
```

不依赖掘金终端，直接用 `history()` API 拉数据模拟回测。
